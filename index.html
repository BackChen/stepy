jQuery Stepy is a plugin based on FormToWizard that generates a customizable wizard.
With default options:
description one
description two

$('#default').stepy();
With custom labels buttons, validation and titles click:
my description one
my description two
my description three Male Female


$('#custom').stepy({
  backLabel:      'Backward',
  nextLabel:      'Forward',
  validate:       true
});
With just step without form:
image description one
image description two
image description three


$('div#step').stepy({
  includeFinish: false
});
HTML structure:
<form id="custom">
  <fieldset title="Title">
    <legend>description</legend>
    <!-- input fields -->
  </fieldset>

  <fieldset title="Title">
    <legend>description</legend>
    <!-- input fields -->
  </fieldset>

  // and so on..

  <input type="submit" class="finish"/>
</form>
* The fieldset's title is the main title of the step and the legend is the description of it.
Default options:
backLabel:      '< Back',
Change the back button label.
includeFinish:  true,
Include the button with class called '.finish' into the last step.
nextLabel:      'Next >'
Change the next button label.
titleClick:     true,
Active the back and next action in the titles.
validate:       false
Active the jQuery Validation for each step. Depends of jquery.validation.js. A div with class 'error-step' is created automatically.
Public functions: *
$.fn.stepy.step(2, 'default');
Go to step 2 with ID called 'default' later. If validation is enabled this feature should come after the call of validate.
* All the functions that do not pass the parameter containerId will be applied to the last container used by the plugin.
